# Week 2 - Lab 2

## Before run

- install rust with [rustup.rs](https://rustup.rs/)

## How to run

- `cargo run` open port [localhost:8000](http://localhost:8000)

## Tools 

- Language : Rust
- Framework: Rocket
- Database : CockroachDB
- OS       : Ubuntu
