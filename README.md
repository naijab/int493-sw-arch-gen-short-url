# homework-2-kotlin

## Must install
- docker, docker-compose

## how to run
- run `docker-compose up -d --build`
- open http://localhost
- run `docker-compose ps` for status

## how to build
- run `./gradlew clean build` or windows use `./gradlew.bat clean build`

## how to ab test
1. edit **ab_test.sh** file
2. run `./ab_test.sh`
